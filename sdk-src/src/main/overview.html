The ALM Octane REST API Java SDK.

To create a new instance use the {@link com.microfocus.adm.almoctane.sdk.Octane} class. Once an Octane instance has been created the shared space and workspace contexts can be added. See the various documentation for more information.

To create a new instance use the following example:

Octane octane = new Octane.Builder(new SimpleUserAuthentication("user", "pass")).build();
